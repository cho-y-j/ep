# docs/details

`/wrap` 슬래시 커맨드가 **세션·주제별 상세 기록**을 남기는 디렉터리입니다.

## 규칙
- 파일명: `YYYY-MM-DD-<주제>.md` (예: `2026-07-10-local-test-env.md`)
- 각 파일 구성:
  1. **배경 / 목표** — 왜 했는지, 성공 기준
  2. **결정 · 구현** — 무엇을 어떻게 (`파일:줄` 인용)
  3. **검증** — 실제 실행한 명령과 결과
  4. **남은 작업** — 후속/미결
- 같은 주제 파일이 있으면 덮어쓰지 말고 이어서 갱신합니다.

`docs/SUMMARY.md` 는 여기 상세들의 최신 요약을, `docs/TODO.md` 는 여기서 파생된 미결 작업을 모읍니다.
